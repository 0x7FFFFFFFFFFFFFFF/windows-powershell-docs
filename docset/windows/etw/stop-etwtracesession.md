# Stop-EtwTraceSession

## SYNOPSIS
Stops the specified ETW session.

## SYNTAX

### ByName (Default)
```
Stop-EtwTraceSession [-Name] <String[]> [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob]
 [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject (cdxml)
```
Stop-EtwTraceSession -InputObject <CimInstance[]> [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>]
 [-AsJob] [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The **Stop-EtwTraceSession** cmdlet stops the specified Event Tracing for Windows (ETW) session.
